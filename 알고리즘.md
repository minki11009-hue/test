## DFS 과정

```
     1
   /   \
  2     3
 / \   
4   5
```

### 단계별 설명

**1단계: 1 방문**
```
stack = [1]
1 꺼냄 → 1의 자식들(2, 3) 스택에 넣음
stack = [2, 3]
```

**2단계: 2 방문** (가장 위 = 최근 것)
```
stack = [2, 3]
2 꺼냄 → 2의 자식들(4, 5) 스택에 넣음
stack = [4, 5, 3]
```

**3단계: 4 방문** (자식의 자식에게 기회!)
```
stack = [4, 5, 3]
4 꺼냄 → 자식 없음
stack = [5, 3]
```

**4단계: 5 방문**
```
stack = [5, 3]
5 꺼냄 → 자식 없음
stack = [3]
```

**5단계: 3 방문** (이제야 1의 오른쪽 자식)
```
stack = [3]
3 꺼냄
```

## 네 말이 정확히 맞아

1. **부모의 자식을 스택에 넣음**
2. **자식들에게 먼저 기회** (스택 위에 있으니까)
3. **자식의 자식에게도 계속 기회** (끝까지 들어감)
4. **막다른 길 → 되돌아가서** 다른 형제들 탐색

**"깊이 우선"인 이유 = 자식의 자식의 자식... 끝까지 가니까!**
